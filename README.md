# Cyber Security Hub — Analytical Forensic Workspace

## 📝 Description
Cyber Security Hub is a responsive, dark-themed Multi-Page Application (MPA) engineered entirely with vanilla frontend web primitives. It serves as a central dashboard for security analysts and students, providing a streamlined workspace to explore cyber forensic analytics parameters, audit digital evidence workflows, and review security threat definitions without heavy framework overhead.

## 🎯 Objective
The objective of this project is to build a high-performance, lightweight reference dashboard for security concepts and cyber forensic audit tracking. By utilizing pure native web technologies, the project demonstrates how to handle data visualization layouts, search filtering, and cross-page parameter synchronization efficiently on the client side.

## 🚀 Key Features
* **Security Control Center:** A premium main landing hub that presents key security metrics, system status indicators, and quick navigation modules.
* **Dynamic Threat & Tool Filter:** An instantaneous client-side filtering system that allows users to sort through forensic tools or threat categories seamlessly by category.
* **Forensic Parameter Routing:** Uses native browser query strings (`URLSearchParams`) to carry specific incident IDs or tool profiles over to a dedicated sub-page for deep layout inspection.
* **Interactive Alert Overlay:** A modal popup system designed to simulate real-time incident responses, vulnerability disclosures, or training media previews without disrupting the main view.
* **Optimized Minimalist UI:** Built with strict CSS asset boundaries, custom dark-mode hover states, and a clean typography system to match modern security operations center (SOC) dashboards.

## 🛠️ Tech Stack Used
* **HTML5:** Semantic document structure for data tables and logs.
* **CSS3:** Custom cybersecurity dark-theme, grid system, and interactive animations.
* **Vanilla JavaScript:** Event listeners, array filtering logic, and native query string processing.
